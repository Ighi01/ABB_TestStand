I2C Free Bus Command

<i2c_free_bus/>

Free the Aardvark I2C subsystem or the Promira I2C subsystem from a held bus condition (e.g., no stop ).

Parameters

None

Details

If the Aardvark I2C subsystem or the Promira I2C subsystem had executed a master transaction and is holding the bus due to a previous nostop flag, this function will issue the stop command and free the bus.

Please see the Aardvark I2C/SPI Adapter User Manual or the Promira Serial Platform User Manual for more information.
