Control Center Serial

SPI Bitrate Command

<spi_bitrate khz="khz"/>

Set the SPI bitrate in kilohertz.

Parameters

khz
the requested bitrate in khz

Details

The power-on default bitrate is 1000 kHz.

Only certain discrete bitrates are supported by the Aardvark adapter or the Promira Serial Platform. As such, this actual bitrate set will be less than or equal to the requested bitrate unless the requested value is less than 125 kHz, in which case the Aardvark adapter or the Promira Serial Platform will default to 125 kHz.

The actual bitrate is returned in the Transaction Log.

Please see the Aardvark I2C/SPI Adapter User Manual or the Promira Serial Platform User Manual for more information.
